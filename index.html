    Django Junior Training Documentation¶

    Complete Guide: From Scratch to Production¶

    üìö Documentation Structure¶

    This comprehensive Django training is divided into multiple modules for easy learning:

    Part 1: Introduction & Setup¶

    • 01_Introduction_and_Setup.md - Django basics, MVT architecture, and environment setup

    Part 2: Function-Based Views (FBV)¶

    • 02_FBV_Models_and_Database.md - Models, migrations, and database operations
    • 03_FBV_Views_and_URLs.md - Function-based views and URL routing
    • 04_FBV_Templates_and_Forms.md - Templates, forms, and frontend
    • 05_FBV_Complete_Project.md - Complete Task Management System with FBV

    Part 3: Class-Based Views (CBV)¶

    • 06_CBV_Introduction.md - Understanding Class-Based Views
    • 07_CBV_Generic_Views.md - Generic views and mixins
    • 08_CBV_Complete_Project.md - Refactoring to Class-Based Views

    üéØ Learning Path¶

    For Complete Beginners:¶

    1. Start with Part 1 (Introduction & Setup)
    2. Follow Part 2 (Function-Based Views) - Build the complete project
    3. Move to Part 3 (Class-Based Views) - Refactor the same project

    For Those with Basic Django Knowledge:¶

    1. Quick review of Part 1
    2. Focus on Part 2 and Part 3 - Compare both approaches
    3. Practice building projects with both methods

    üìù What You'll Learn¶

    Core Django Concepts:¶

    • MVT (Model-View-Template) architecture
    • Django ORM and database operations
    • URL routing and views
    • Template system and template inheritance
    • Forms and validation
    • User authentication and authorization
    • Static files and media handling
    • Admin interface customization

    Function-Based Views:¶

    • Creating views as Python functions
    • Request/response handling
    • Decorators for authentication
    • Manual form handling
    • Custom logic implementation

    Class-Based Views:¶

    • Object-oriented view design
    • Generic views (ListView, DetailView, etc.)
    • Mixins for code reuse
    • Method overriding
    • Advanced CBV patterns

    Advanced Concepts:¶

    • When to use FBV vs CBV
    • Code organization and patterns
    • Security considerations
    • Performance optimization techniques
